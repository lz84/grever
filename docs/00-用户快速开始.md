# Grever 快速开始

3 步即可启动 Grever 并体验演示数据。

## 前置条件

- 已安装 [Docker](https://www.docker.com/) 和 Docker Compose
- 已安装 Python 3.10+（用于运行初始化脚本）

## 第 1 步：Docker 部署

在项目根目录执行：

```bash
docker-compose up -d
```

这将启动后端（FastAPI，端口 8096）和前端（Vue3，端口 5173）两个容器。

## 第 2 步：初始化数据库

等待容器启动后，运行初始化脚本并灌入演示数据：

```bash
python scripts/init_grever.py --seed
```

该脚本会：
- 创建数据库表结构
- 灌入演示数据（软件开发场景）

如需清空并重置数据库，使用 `--reset` 参数：

```bash
python scripts/init_grever.py --reset
```

## 第 3 步：访问系统

浏览器打开：

```
http://localhost:5173
```

即可看到 Grever 界面和演示数据。
