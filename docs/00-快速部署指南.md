# Nexus 快速部署指南

**版本**: v0.1
**日期**: 2026-05-27
**目标**: 从零到跑起来，10 分钟

---

## 环境要求

| 依赖 | 最低版本 | 说明 |
|------|---------|------|
| Python | 3.10+ | 后端运行时 |
| Node.js | 18+ | 前端开发服务器 |
| npm | 9+ | 包管理 |
| Git | 任意 | 拉代码 |

> 不需要 Docker、MySQL、Redis。SQLite 内置，开箱即用。

---

## 第一步：拉代码

```bash
git clone <仓库地址>
cd agents-nexus
```

---

## 第二步：启动后端

```bash
cd packages/server
pip install -r ../../config/requirements.txt
python run_server.py
```

启动成功标志：

```
INFO:     Uvicorn running on http://0.0.0.0:8097
```

验证：浏览器打开 `http://localhost:8097/health`，看到 `{"status":"ok"}`。

---

## 第三步：启动前端

```bash
cd packages/ui
npm install
npm run dev
```

启动成功标志：

```
VITE ready in xxx ms
➜  Local:   http://localhost:5173/
```

---

## 第四步：打开浏览器

访问 `http://localhost:5173/`

---

## 配置文件

| 文件 | 必填 | 说明 |
|------|------|------|
| `.env` | 否 | LLM API 配置（不填也能用，只是验证引擎跳过 LLM 检查） |
| `packages/ui/.env.development` | 是 | 前端代理地址，默认指向 `http://127.0.0.1:8097` |

---

## 数据库

- **类型**: SQLite
- **位置**: `data/reins.db`
- **大小**: 47 MB（含演示数据）
- **说明**: 首次启动自动建表，无需手动迁移

---

## Agent 说明

系统内置 5 个 Agent（刚子/谷子/麻子/蚊子/扣子），它们由 **OpenClaw Gateway** 管理。

| 情况 | Agent 状态 | 影响 |
|------|-----------|------|
| 有 OpenClaw Gateway 运行 | 在线 | 任务可以自动派发执行 |
| 没有 OpenClaw Gateway | 离线 | 系统正常运行，但任务不会执行 |

试用阶段如果暂时跑不了 OpenClaw，**不影响体验系统功能**：可以创建目标、场景、项目、任务，只是任务状态不会自动流转。

---

## 常见问题

| 问题 | 解法 |
|------|------|
| 端口 8097 被占用 | 杀掉占用进程，或修改 `.env` 换端口 |
| 前端连不上后端 | 确认 `packages/ui/.env.development` 中的 URL 正确 |
| npm install 报错 | 确认 Node.js ≥ 18 |
| 页面白屏 | 检查后端是否正常运行，浏览器控制台看 API 报错 |
