## 一、设计目标

### 1.1 背景

当前 Agent 注册和触发流程存在以下问题：

| 问题 | 现状 | 影响 |
|------|------|------|
| **注册表单硬编码** | `AgentRegisterModal.tsx` 写死一套字段（name + 4 维能力 + trigger_mode） | 无法注册 Dify/Coze 等不同平台的 Agent |
| **后端 schema 硬编码** | `_agents_register_routes.py` 的 `AgentRegister` Pydantic 模型固定字段 | 新增平台需改后端代码 |
| **触发方式绑死** | `task_runner.py` 硬编码 `AGENT_UUID_TO_CODE` 映射 + OpenClaw CLI spawn | 无法调度 Dify/Coze 等平台上的 Agent |
| **平台配置无存储** | agents 表没有 platform_type 和平台专属配置字段 | 无法区分不同平台的 Agent |
| **同步异步不统一** | OpenClaw 是同步 spawn + poll，其他平台可能是异步 | 上层调用方需处理多种模式 |

### 1.2 设计原则

1. **门面层统一**：对外（其他域、API 调用方）只暴露一套标准接口
2. **适配层可插拔**：新增平台只需实现适配器接口 + 注册，不改核心代码
3. **前端动态渲染**：不同平台的注册表单由后端 schema 驱动，不是硬编码
4. **异步优先**：所有派发统一返回 DispatchResult，上层不关心同步/异步差异
5. **能不创建就不创建**：注册 = 关联（登记 Agent 的存在和调用方式），不是创建
6. **零破坏迁移**：现有 agents 表和 API 兼容，渐进式改造

## 二、概念澄清

### 2.1 持久 Agent vs 能力节点

Nexus 管理的 Agent 分两类，注册语义不同：

| 类型 | 注册的是什么 | status/load 含义 | 示例 |
|------|-------------|-----------------|------|
| **持久 Agent** | 一个活着的 Agent 实例 | 描述该 Agent 的真实状态 | OpenClaw/Dify/Coze |
| **能力节点（Node）** | 一台可以 spawn 会话的能力 | 描述该节点是否还能接新任务 | Claude Code/Codex/Copilot |

**类比**：
- 持久 Agent = 注册了一只"狗"（活着的 Agent，有自己的状态）
- 能力节点 = 注册了一个"狗窝"（可以派新会话的能力，状态描述的是节点容量）

对能力节点：
- `status` 描述的是节点本身是否在线（CLI 可用、环境变量已配置）
- `load` / `current_tasks` 描述的是节点上同时运行的会话数
- `max_concurrent_tasks` 是关键——这台机器同时能跑几个会话

### 2.2 认证管理策略

不同平台的认证方式分为两类：

| 认证模式 | 说明 | 适用平台 |
|---------|------|---------|
| **Nexus 管理密钥** | api_key 存入 config_json（加密存储），Nexus 持有并传递 | Dify, Coze |
| **记录认证来源** | Nexus 不存密钥，只记录密钥来源（环境变量/配置文件），dispatch 时确保继承 | Claude Code, Codex, Copilot, OpenClaw |

**CLI 平台（Claude Code/Codex/Copilot/OpenClaw）**：这些工具的 API Key 是宿主环境的安全边界，Nexus 不集中存储密钥，而是在 `config_json` 中声明需要哪些环境变量。dispatch 时通过环境变量继承确保 CLI 能认证。

**platform_type 枚举值**：

### 3.2 agents_config 表（新增）

```sql
CREATE TABLE agents_config (
    agent_id    VARCHAR(32) PRIMARY KEY,  -- FK → agents.id
    platform_type VARCHAR(32) NOT NULL,   -- 冗余，用于快速过滤（ORM 层保证一致性）
    config_json TEXT NOT NULL DEFAULT '{}', -- 平台专属配置
    created_at  DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    updated_at  DATETIME,
    FOREIGN KEY (agent_id) REFERENCES agents(id) ON DELETE CASCADE
);
```

**为什么用 config_json 而不是列存**：

- 列存方案每加一个平台就要 ALTER TABLE 加列，且大量 NULL 列浪费
- JSON 方案加新平台只需加前端组件 + 适配器，不改表结构
- **前端不暴露 JSON**：后端通过 `get_registration_fields()` 暴露 schema，前端按 schema 渲染表单字段
- **后端严格校验**：注册时按平台 schema 校验 config_json，不是随便塞

**冗余列说明**：`agents_config.platform_type` 是从 `agents.platform_type` 冗余过来的。保留原因是：查"所有 Dify Agent 的配置"时无需 JOIN agents 表（config_json 很长，JOIN 会把 JSON 拉进临时表）。ORM 层保证两张表的 platform_type 始终一致。

**各平台 config_json 示例**：

## 四、适配层设计

### 4.1 核心概念

```
FieldDef — 字段定义（key/label/type/required/validation + auth_source 标识）
  ↓
BaseAgentAdapter — 适配器抽象基类（get_registration_fields + register + dispatch + ...）
  ↓
OpenClawAdapter / DifyAdapter / CozeAdapter / ClaudeCodeAdapter / CodexAdapter / CopilotAdapter
  ↓
AgentAdapterRegistry — 注册表（工厂 + 路由）
```

### 4.2 FieldDef — 字段定义

```python
@dataclass
class FieldDef:
    """注册表单字段定义"""
    key: str                          # 字段名，如 'api_key'
    label: str                        # 前端显示名，如 'API Key'
    type: str                         # string | number | password | url | select | textarea
    required: bool = False            # 是否必填
    placeholder: str = ""             # 输入框占位符
    description: str = ""             # 字段说明
    default: Any = None               # 默认值
    options: List[Dict[str, str]] = None  # select 类型专用
    validation: Dict[str, Any] = None     # 校验规则
    sensitive: bool = False               # 是否敏感字段（True → DB 加密存储）
```

新增 `sensitive` 字段：标记需要加密存储的字段（如 api_key）。后端在写入 agents_config 前自动加密，API 返回时自动脱敏。

### 4.3 BaseAgentAdapter — 抽象基类

```python
# agent_service/adapters/base.py
from abc import ABC, abstractmethod
from typing import List, Dict, Any, Optional

@dataclass
class DispatchResult:
    dispatch_id: str          # 派发 ID
    status: str               # "accepted" | "completed" | "failed"
    result: Optional[str] = None   # 执行结果（同步模式有值）
    error: Optional[str] = None    # 错误信息
    estimated_seconds: int = 0     # 预估耗时（秒）

@dataclass
class TaskDispatch:
    task_id: str
    title: str
    description: str
    context_md: Optional[str] = None
    acceptance_criteria: Optional[str] = None
    timeout_seconds: int = 300

class BaseAgentAdapter(ABC):
    """所有 Agent 平台适配器必须实现的抽象接口"""

    @property
    @abstractmethod
    def platform_type(self) -> str:
        """平台类型标识，如 'openclaw', 'dify', 'coze'"""
        pass

    @property
    def platform_label(self) -> str:
        """平台显示名，如 'OpenClaw', 'Dify'"""
        return self.platform_type

    @abstractmethod
    def get_registration_fields(self) -> List[FieldDef]:
        """
        返回该平台的注册字段定义
        
        后端暴露为 API：GET /api/v1/agent-platforms/{type}/registration-schema
        前端拿这个动态渲染表单
        """
        pass

    @abstractmethod
    async def register(self, agent_id: str, name: str,
                       config: Dict[str, Any]) -> str:
        """
        注册/关联 Agent
        
        - 验证 platform_config 是否可用（如 Dify 试调一次 API，CLI 验证 --version）
        - 验证认证凭据（Nexus 管理的密钥试调 / 环境变量的 CLI 平台检查 env 存在）
        - 返回 agent_id
        """
        pass

    @abstractmethod
    async def unregister(self, agent_id: str, config: Dict[str, Any]) -> bool:
        """注销 Agent"""
        pass

    @abstractmethod
    async def heartbeat(self, agent_id: str, config: Dict[str, Any]) -> bool:
        """
        心跳检查
        
        - OpenClaw: CLI health check
        - Dify: GET {base_url}/health + 可选 workflow test
        - Coze: 通过最近消息判断活跃
        - Claude Code/Codex/Copilot: 验证 CLI 可用 + 环境变量存在
        """
        pass

    @abstractmethod
    async def dispatch(self, agent_id: str, config: Dict[str, Any],
                       task: TaskDispatch) -> DispatchResult:
        """
        派发任务到 Agent
        
        - OpenClaw: openclaw sessions spawn --agent xxx --prompt xxx
        - Dify: POST {base_url}/v1/workflows/run
        - Coze: POST https://api.coze.com/v1/chat
        - Claude Code: claude -p "prompt" --allow-all --yolo --work-dir xxx
        - Codex: codex run -p "prompt" --work-dir xxx
        - Copilot: copilot -p "prompt" --allow-all --yolo --add-dir xxx
        """
        pass

    @abstractmethod
    async def get_result(self, dispatch_id: str,
                         config: Dict[str, Any]) -> Optional[TaskResult]:
        """
        获取任务执行结果（异步模式专用）
        """
        pass

    def is_async_native(self) -> bool:
        """
        该平台是否原生异步（True = dispatch 立即返回，需轮询 get_result）
        默认 False（同步），子类 override
        """
        return False

    def is_session_based(self) -> bool:
        """
        该平台是否按会话实例化（能力节点 vs 持久 Agent）
        True: Claude Code, Codex, Copilot
        False: OpenClaw, Dify, Coze
        """
        return False

    def validate_config(self, config: Dict[str, Any]) -> List[str]:
        """校验 platform_config 是否符合该平台的 schema"""
        errors = []
        fields = self.get_registration_fields()
        for field in fields:
            if field.required and field.key not in config:
                errors.append(f"缺少必填字段: {field.label} ({field.key})")
            elif field.key in config and field.validation:
                value = str(config[field.key])
                if field.validation.get("min_length") and len(value) < field.validation["min_length"]:
                    errors.append(f"{field.label} 长度不足")
                if field.validation.get("max_length") and len(value) > field.validation["max_length"]:
                    errors.append(f"{field.label} 长度超限")
                if field.validation.get("pattern") and not re.match(field.validation["pattern"], value):
                    errors.append(f"{field.label} 格式不正确")
                if field.validation.get("url") and not self._is_valid_url(value):
                    errors.append(f"{field.label} 不是有效的 URL")
        return errors
```

### 4.5 AgentAdapterRegistry — 适配器注册表

```python
class AgentAdapterRegistry:
    """Agent 适配器注册表 — 工厂 + 运行时路由"""

    def __init__(self):
        self._adapters: Dict[str, BaseAgentAdapter] = {}

    def register(self, adapter: BaseAgentAdapter):
        self._adapters[adapter.platform_type] = adapter

    def get(self, platform_type: str) -> BaseAgentAdapter:
        if platform_type not in self._adapters:
            raise KeyError(f"Adapter for '{platform_type}' not registered")
        return self._adapters[platform_type]

    def has(self, platform_type: str) -> bool:
        return platform_type in self._adapters

    def list_platforms(self) -> List[Dict[str, Any]]:
        """返回所有已注册的平台（供前端平台选择器）"""
        return [
            {
                "type": a.platform_type,
                "label": a.platform_label,
                "available": True,
                "is_session_based": a.is_session_based(),
            }
            for a in self._adapters.values()
        ]

    def auto_detect(self) -> str:
        return "openclaw"
```

## 五、门面层设计

### 5.1 AgentFacade — 主门面

```python
# agent_service/facade.py
class AgentFacade:
    """
    Agent 门面层 — 对外唯一入口
    
    所有其他服务（Reins 调度/Grasp 认知/Evo 进化）和 API 路由
    只跟这个门面打交道，不直接接触适配器。
    """

    def __init__(self, registry: AgentAdapterRegistry, db):
        self._registry = registry
        self._db = db

    async def register(self, request: AgentRegisterRequest) -> AgentRegistered:
        """
        注册 Agent
        
        流程：
        1. 获取对应平台的 adapter
        2. 用 adapter.get_registration_fields() 校验 platform_config
        3. 调 adapter.register() 在目标平台验证连通性
        4. 写 agents 表（platform_type + 基础字段）
        5. 写 agents_config 表（platform_config，敏感字段加密）
        6. 返回注册结果
        """
        adapter = self._registry.get(request.platform_type)
        config = request.platform_config or {}
        errors = adapter.validate_config(config)
        if errors:
            raise ValidationError(f"平台配置校验失败: {'; '.join(errors)}")

        agent_id = await adapter.register(
            agent_id=request.agent_id,
            name=request.name,
            config=config,
        )

        self._save_agent_to_db(request)
        # 敏感字段自动加密后写入 agents_config
        self._save_agent_config(agent_id, request.platform_type, config)
        return AgentRegistered(agent_id=agent_id, status="online")

    async def unregister(self, agent_id: str, reason: str = None) -> bool:
        agent = self._get_agent_from_db(agent_id)
        adapter = self._registry.get(agent.platform_type)
        config = self._get_agent_config(agent_id)
        success = adapter.unregister(agent_id, config)
        if success:
            self._delete_agent_from_db(agent_id)
        return success

    async def heartbeat(self, agent_id: str, status: Dict = None) -> HeartbeatResult:
        agent = self._get_agent_from_db(agent_id)
        adapter = self._registry.get(agent.platform_type)
        config = self._get_agent_config(agent_id)
        alive = await adapter.heartbeat(agent_id, config)
        if alive:
            self._update_agent_heartbeat(agent_id)
        return HeartbeatResult(agent_id=agent_id, alive=alive)

    async def dispatch(self, agent_id: str, task: TaskDispatch) -> DispatchResult:
        """派发任务"""
        agent = self._get_agent_from_db(agent_id)
        adapter = self._registry.get(agent.platform_type)
        config = self._get_agent_config(agent_id)
        result = await adapter.dispatch(agent_id, config, task)
        self._update_task_assignment(task.task_id, agent_id, result)
        return result

    async def get_result(self, dispatch_id: str, agent_id: str) -> Optional[TaskResult]:
        """获取异步任务结果"""
        agent = self._get_agent_from_db(agent_id)
        adapter = self._registry.get(agent.platform_type)
        config = self._get_agent_config(agent_id)
        return await adapter.get_result(dispatch_id, config)

    def list_platforms(self) -> List[Dict[str, Any]]:
        return self._registry.list_platforms()

    def get_registration_schema(self, platform_type: str) -> Dict[str, Any]:
        """获取指定平台的注册 schema（供前端动态渲染）"""
        adapter = self._registry.get(platform_type)
        return {
            "platform_type": adapter.platform_type,
            "platform_label": adapter.platform_label,
            "is_session_based": adapter.is_session_based(),
            "fields": [
                {
                    "key": f.key, "label": f.label, "type": f.type,
                    "required": f.required, "placeholder": f.placeholder,
                    "description": f.description, "default": f.default,
                    "options": f.options, "validation": f.validation,
                    "sensitive": f.sensitive,
                }
                for f in adapter.get_registration_fields()
            ],
        }
```

### 8.2 异步轮询机制

对于异步平台（Dify/Coze），需要在后台定时轮询结果。采用**指数退避**策略：

```python
class DispatchPoller:
    """异步派发结果轮询器"""
    
    INITIAL_POLL_INTERVAL = 10    # 初始 10 秒
    MAX_POLL_INTERVAL = 120       # 最大 120 秒
    BACKOFF_MULTIPLIER = 2        # 每次翻倍
    DEFAULT_TIMEOUT = 1800        # 默认 30 分钟
    
    async def poll(self):
        pending = self._get_pending_dispatches()
        for dispatch in pending:
            # 获取该任务的超时配置（优先用 task 传入的，否则用默认值）
            timeout = dispatch.timeout_seconds or self.DEFAULT_TIMEOUT
            
            if dispatch.elapsed_seconds > timeout:
                self._fail_task(dispatch.task_id, "dispatch timeout")
                continue
            
            agent = self._get_agent(dispatch.agent_id)
            adapter = self._registry.get(agent.platform_type)
            config = self._get_agent_config(dispatch.agent_id)
            
            result = await adapter.get_result(dispatch.dispatch_id, config)
            if result:
                self._complete_task(dispatch.task_id, result)
            else:
                # 指数退避：10s → 20s → 40s → 80s → 120s（cap）
                next_interval = min(
                    dispatch.poll_interval * self.BACKOFF_MULTIPLIER,
                    self.MAX_POLL_INTERVAL,
                )
                dispatch.poll_interval = next_interval
```

## 十一、安全与加密

### 11.1 敏感字段加密

`config_json` 中标记为 `sensitive: true` 的字段（如 api_key、pat）在写入数据库前需要加密：

```python
class ConfigEncryptor:
    """config_json 敏感字段加密"""
    
    def encrypt(self, config: Dict, fields: List[FieldDef]) -> Dict:
        """加密敏感字段"""
        result = dict(config)
        for field in fields:
            if field.sensitive and field.key in result:
                result[field.key] = self._encrypt_value(result[field.key])
        return result
    
    def decrypt(self, config: Dict, fields: List[FieldDef]) -> Dict:
        """解密敏感字段（dispatch 等需要原始值时调用）"""
        result = dict(config)
        for field in fields:
            if field.sensitive and field.key in result:
                result[field.key] = self._decrypt_value(result[field.key])
        return result
```

加密方案：使用 Fernet 对称加密（密钥通过环境变量注入，不存 DB）。

### 11.2 API 脱敏规则

返回 agents_config 给前端时，敏感字段自动脱敏：

| 原始值 | 脱敏后 |
|--------|--------|
| `app-abc123def456` | `app-****def456` |
| `pat_xyz789abc012` | `pat_****abc012` |

脱敏规则：保留前缀 4 个字符 + `****` + 后缀 6 个字符。
