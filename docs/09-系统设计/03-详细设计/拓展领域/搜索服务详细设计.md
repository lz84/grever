## 1. 概述

### 1.1 设计目标

Nexus 搜索服务是 Reach（达）系统的核心组件，提供全局搜索能力，支持跨目标（Goal）、项目（Project）、任务（Task）的快速检索。核心设计原则为 **轻量级全文搜索**：基于关键词匹配，返回结构化搜索结果。

### 1.2 核心职责

| 职责 | 说明 |
|------|------|
| 全局搜索 | 支持 Goal/Project/Task 的关键词搜索 |
| 分类返回 | 按类型分组返回搜索结果 |
| 结果限制 | 每类结果数量限制，避免过多返回 |
| 模糊匹配 | 大小写不敏感的部分匹配 |

## 2. 核心流程

### 2.1 搜索执行流程

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                           搜索执行流程                                        │
└──────────────────────────────────────────────────────────────────────────────┘

Step 1: 关键词预处理
  q.strip().lower()
         │
         ▼
Step 2: 搜索 Goals
  查询 goals 表
  匹配 title / description
         │
         ▼
Step 3: 搜索 Projects
  查询 projects 表
  匹配 name / description
         │
         ▼
Step 4: 搜索 Tasks
  查询 tasks 表
  匹配 title / description
         │
         ▼
Step 5: 汇总结果
  合并各类型结果
  统计 total
         │
         ▼
Step 6: 返回响应
  {goals: [], projects: [], tasks: [], total: N}
```

## 3. 数据模型

### 3.1 搜索请求

| 字段 | 类型 | 说明 |
|------|------|------|
| q | str | 搜索关键词（必填） |
| limit | int | 每类返回数量上限（默认 10，最大 50） |

### 3.2 搜索响应

```json
{
  "query": "搜索关键词",
  "goals": [
    {
      "id": "goal-xxx",
      "title": "目标标题",
      "description": "目标描述",
      "status": "in_progress",
      "type": "goal"
    }
  ],
  "projects": [...],
  "tasks": [...],
  "total": 30
}
```

### 3.3 单个结果字段

| 字段 | 类型 | 说明 |
|------|------|------|
| id | str | 资源 ID |
| title/name | str | 资源标题/名称 |
| description | str | 资源描述 |
| status | str | 资源状态 |
| type | str | 资源类型：goal/project/task |
| goal_id | str | 所属目标（仅 Project） |
| project_id | str | 所属项目（仅 Task） |
| assigned_agent | str | 分配 Agent（仅 Task） |

## 4. 搜索算法

### 4.1 匹配规则

```python
# 大小写不敏感
query = q.strip().lower()

# 部分匹配（substring）
if query in (g.title or "").lower():
    matched.append({...})

if query in (g.description or "").lower():
    matched.append({...})
```

### 4.2 结果限制

```python
# 每类最多返回 limit 条
if len(matched) >= limit:
    break
```

## 7. 错误处理

### 7.1 错误码定义

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| 400 | 搜索词为空 | 返回 "搜索词不能为空" |
| 500 | 服务器内部错误 | 记录日志，返回空结果 |

### 7.2 异常处理

```python
@router.get("/api/v1/search")
def global_search(q: str, limit: int):
    if not q or not q.strip():
        raise HTTPException(status_code=400, detail="搜索词不能为空")
    
    # 各模块搜索独立异常处理
    try:
        goals = db.query(GoalModel).all()
        # ...
    except Exception as e:
        logger.error(f"[Search] Goals search error: {e}")
```

