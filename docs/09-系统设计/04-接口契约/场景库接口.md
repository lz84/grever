## 一、概述

### 1.1 接口范围

本文档定义场景库相关的所有 API 接口，包括场景模板 CRUD、场景匹配、场景实例化、条件分支评估和场景反馈。

### 1.2 核心概念

| 概念 | 说明 |
|------|------|
| **场景模板** | 静态的解决方案蓝图，事前定义所有可能执行路径的全集（fullset） |
| **场景实例** | 从模板实例化到具体 Goal 的执行计划 |
| **实例化** | 从模板裁剪出初始 Project + Task 的过程 |
| **条件分支** | 根据条件或人工输入决定执行路径的决策节点 |

## 二、接口清单总表

### 2.1 场景模板 CRUD

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 创建场景模板 | POST | `/api/v1/scenarios` | 创建新场景模板 |
| 查询场景列表 | GET | `/api/v1/scenarios` | 列表查询 |
| 查询场景详情 | GET | `/api/v1/scenarios/{scenario_id}` | 获取单个场景详情 |
| 更新场景模板 | PUT | `/api/v1/scenarios/{scenario_id}` | 更新场景信息 |
| 删除场景模板 | DELETE | `/api/v1/scenarios/{scenario_id}` | 软删除或硬删除 |
| 查询场景 fullset | GET | `/api/v1/scenarios/{scenario_id}/fullset` | 获取场景完整集 |
| 更新场景 fullset | PUT | `/api/v1/scenarios/{scenario_id}/fullset` | 更新场景完整集 |
| 查询版本历史 | GET | `/api/v1/scenarios/{scenario_id}/versions` | 获取版本列表 |

### 2.2 场景实例化

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 场景匹配 | POST | `/api/v1/scenarios/match` | 根据 Goal 描述匹配场景 |
| 实例化场景 | POST | `/api/v1/goals/{goal_id}/instantiate-scenario` | 实例化到 Goal |
| 查询场景实例 | GET | `/api/v1/goals/{goal_id}/scenario-instance` | 获取 Goal 的实例 |
| 查询待触发条件 | GET | `/api/v1/goals/{goal_id}/pending-conditions` | 获取待触发条件 |
| 触发条件评估 | POST | `/api/v1/scenario-conditions/{cond_id}/evaluate` | 手动触发条件评估 |
| 创建自定义 Project | POST | `/api/v1/projects/custom-create` | 创建自定义 Project |

### 2.3 场景反馈

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 提交场景反馈 | POST | `/api/v1/scenarios/{scenario_id}/feedback` | 提交执行反馈 |
| 查询场景反馈 | GET | `/api/v1/scenarios/{scenario_id}/feedback` | 获取反馈历史 |

### 2.4 场景扩展功能（新增）

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 为目标创建场景 | POST | `/api/v1/scenarios/create-for-goal/{goal_id}` | 根据目标创建场景 |
| 为目标匹配场景 | POST | `/api/v1/scenarios/match-for-goal/{goal_id}` | 根据目标匹配场景 |

| 收藏场景列表 | GET | `/api/v1/scenarios/starred` | 获取收藏的场景列表 |
| 收藏/取消收藏 | POST | `/api/v1/scenarios/{scenario_id}/star` | 切换收藏状态 |

## 三、接口详情

### 3.1 创建场景模板

```
POST /api/v1/scenarios
Content-Type: application/json

Request Body:
{
  "name": "化工厂危化品泄漏应急预案",
  "description": "化工园区危化品泄漏标准响应流程",
  "industry": "chemical-emergency",
  "version": "1.0.0",
  "goal_tags": {
    "business": ["emergency_response"],
    "professional": ["incident_command"],
    "technical": [],
    "management": ["coordination", "task_dispatch"]
  },
  "fullset": {
    "projects": [
      {
        "name": "应急响应",
        "type": "mandatory",
        "capability_tags": { "business": ["emergency_response"] },
        "tasks": [
          { "name": "启动应急预案", "type": "mandatory", "capability_tags": {} }
        ]
      },
      {
        "name": "人员疏散",
        "type": "conditional",
        "condition": {
          "type": "human_decision",
          "prompt": "根据风向，是否需要疏散周边人员？",
          "options": ["立即疏散", "暂缓监测", "不需要"],
          "branches": {
            "立即疏散": { "next_projects": ["疏散执行"] },
            "暂缓监测": { "next_projects": ["持续监测"] }
          }
        }
      }
    ]
  }
}

Response 201:
{
  "id": "sce-001",
  "name": "化工厂危化品泄漏应急预案",
  "industry": "chemical-emergency",
  "version": "1.0.0",
  "status": "draft",
  "created_at": 1716800000
}
```


## 四、fullset Schema

```json
{
  "goal_tags": {
    "business": [],
    "professional": [],
    "technical": [],
    "management": []
  },
  "projects": [
    {
      "name": "string",
      "type": "mandatory | conditional",
      "capability_tags": {},
      "tasks": [
        {
          "name": "string",
          "type": "mandatory | conditional",
          "capability_tags": {},
          "condition": {
            "type": "none | auto_eval | human_decision | human_input",
            "expression": "string (auto_eval)",
            "prompt": "string (human_*)",
            "options": ["string"],
            "branches": {}
          }
        }
      ]
    }
  ]
}
```

## 五、错误码定义

| HTTP 状态码 | 错误码 | 说明 |
|------------|--------|------|
| 400 | `invalid_fullset` | fullset 格式错误 |
| 400 | `instance_create_failed` | 实例化失败 |
| 404 | `scenario_not_found` | 场景不存在 |
| 404 | `instance_not_found` | 场景实例不存在 |
| 409 | `scenario_name_exists` | 场景名称已存在 |
