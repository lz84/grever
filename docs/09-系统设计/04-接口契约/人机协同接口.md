## 一、概述

### 1.1 接口范围

本文档定义 Human-in-the-Loop（HITL）人机协同相关的所有 API 接口，包括待裁决/待审批/待输入请求的查询、提交响应、权限检查、terminate 和 takeover 操作。

### 1.2 HITL 类型说明

| 类型 | 代码 | 说明 |
|------|------|------|
| **人工裁决** | `disputed` | 验证循环超过 3 次仍不通过，需要人类裁决 |
| **人工审批** | `waiting_human` | 任务等待人类确认后才能继续 |
| **人工协助** | `human_input` | 任务需要人类提供信息才能继续执行 |

### 1.3 状态说明

| 状态 | 说明 |
|------|------|
| `pending` | 等待人类响应 |
| `answered` | 人类已响应 |
| `timed_out` | 超时未响应 |
| `cancelled` | 已取消 |

  "action": "done"

  "ruling": "API 端点不可达，请修复后重新提交。",

  "action": "verifying"

{

  "status": "done",
  "ruling_comment_id": "cmt-003",
  "verification_cycle": 0,

- 400: 任务状态不是 disputed
  {"error": "task_not_disputed", "current_status": "review_needed", "detail": "只有 disputed 状态的任务可以提交裁决"}
- 404: 任务不存在
  {"error": "task_not_found", "task_id": "task-xxx"}

## 四、human_input_requests 表扩展字段

Sprint 93/94 扩展后支持 Goal/Project 级别 HITL：

| 字段 | 类型 | 说明 | 必填 |
|------|------|------|------|
| id | TEXT (PK) | 请求 ID | ✅ |
| task_id | TEXT (FK) | 关联任务（Task 级 HITL） | 否 |
| project_id | TEXT (FK) | 关联项目（Project 级 HITL） | 否 |
| goal_id | TEXT (FK) | 关联目标（Goal 级 HITL） | 否 |
| scenario_ref | TEXT | 关联到场景的哪个 condition | 否 |
| title | TEXT | 请求标题 | ✅ |
| description | TEXT | 详细描述 | 否 |
| input_type | TEXT | 输入类型：text/number/select/multiline/decision | ✅ |
| options | TEXT | 选项列表（JSON array，select/decision 类型用） | 否 |
| status | TEXT | 状态：pending/answered/timed_out/cancelled | ✅ |
| response | TEXT | 人类回答 | 否 |
| responder_id | TEXT | 回答者 ID | 否 |
| default_value | TEXT | 超时默认值 | 否 |
| timeout_action | TEXT | 超时行为：use_default/skip_project/skip_task/escalate | 否 |
| timeout_minutes | INTEGER | 超时时间 | 否 |
| branches | TEXT | 答案到分支的映射（JSON） | 否 |
| context | TEXT | 上下文信息（JSON） | 否 |
| created_at | INTEGER | 创建时间 | ✅ |
| updated_at | INTEGER | 更新时间 | 否 |

**互斥约束**：`task_id`、`project_id`、`goal_id` 三个字段只能有一个非空。

## 五、错误码定义

| HTTP 状态码 | 错误码 | 说明 |
|------------|--------|------|
| 400 | `task_not_disputed` | 只有 disputed 状态的任务可以裁决 |
| 400 | `invalid_status` | 任务状态不允许该操作 |
| 400 | `request_timed_out` | human_input 请求已超时 |
| 400 | `invalid_action` | action 必须是 done/in_progress/verifying |
| 404 | `task_not_found` | 任务不存在 |
| 404 | `request_not_found` | human_input 请求不存在 |
| 409 | `already_responded` | 请求已被响应，不能重复提交 |
