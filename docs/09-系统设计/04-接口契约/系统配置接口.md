## 3. 详细接口说明

### 3.1 Settings 模块

#### GET /api/v1/settings/
获取所有设置分类列表。

**响应示例：**
```json
{
  "categories": ["model", "api", "session", "notification"]
}
```

#### GET /api/v1/settings/{category}
获取指定分类下的所有设置项。

**路径参数：**
- `category`（string）：分类名，如 `model`、`api`

**响应示例：**
```json
{
  "category": "model",
  "items": {
    "default_model": "minimax/MiniMax-M2.7",
    "temperature": 0.7,
    "max_tokens": 4096
  }
}
```

#### GET /api/v1/settings/{category}/{key}
获取单个设置项的值。

**路径参数：**
- `category`（string）：分类名
- `key`（string）：设置键名

**响应示例：**
```json
{
  "category": "model",
  "key": "default_model",
  "value": "minimax/MiniMax-M2.7"
}
```

#### PUT /api/v1/settings/{category}/{key}
更新单个设置项。

**路径参数：**
- `category`（string）：分类名
- `key`（string）：设置键名

**请求体：**
```json
{
  "value": "minimax/MiniMax-M2.7-highspeed"
}
```

**响应示例：**
```json
{
  "success": true,
  "category": "model",
  "key": "default_model",
  "value": "minimax/MiniMax-M2.7-highspeed"
}
```

#### PUT /api/v1/settings/{category}/batch
批量更新某分类下的多个设置项。

**请求体：**
```json
{
  "items": {
    "default_model": "minimax/MiniMax-M2.7-highspeed",
    "temperature": 0.8
  }
}
```

**响应示例：**
```json
{
  "success": true,
  "category": "model",
  "updated": 2
}
```

#### GET /api/v1/settings/models
获取可用的模型列表。

**响应示例：**
```json
{
  "models": [
    {"id": "minimax/MiniMax-M2.7", "name": "MiniMax M2.7"},
    {"id": "minimax/MiniMax-M2.7-highspeed", "name": "MiniMax M2.7 高速"},
    {"id": "zhzy/qwen3-30b-a3b-fp8", "name": "Qwen3 30B"}
  ]
}
```

#### GET /api/v1/settings/sessions
获取当前活跃会话列表。

**响应示例：**
```json
{
  "sessions": [
    {"id": "sess_001", "agent": "刚子", "status": "active", "last_active": "2026-05-27T17:00:00Z"}
  ]
}
```

#### POST /api/v1/settings/test-connection
测试外部服务连接（如 API 网关、数据库等）。

**请求体：**
```json
{
  "type": "database",
  "config": {
    "provider": "sqlite",
    "path": "data/reins.db"
  }
}
```

**响应示例：**
```json
{
  "success": true,
  "latency_ms": 12,
  "message": "Connection successful"
}
```

### 3.2 Security 模块

#### GET /api/v1/security/alerts
获取安全告警列表。

**查询参数：**
- `status`（string，可选）：过滤条件，如 `open`、`resolved`
- `limit`（integer，可选）：返回数量，默认 50

**响应示例：**
```json
{
  "alerts": [
    {
      "id": "alert_001",
      "type": "unauthorized_access",
      "severity": "high",
      "message": "检测到来自 192.168.1.100 的异常登录尝试",
      "status": "open",
      "created_at": "2026-05-27T10:30:00Z"
    }
  ],
  "total": 1
}
```

#### GET /api/v1/security/audit/logs
获取审计日志。

**查询参数：**
- `actor`（string，可选）：操作者 ID
- `action`（string，可选）：操作类型
- `from`（datetime，可选）：开始时间
- `to`（datetime，可选）：结束时间
- `limit`（integer，可选）：返回数量，默认 100

**响应示例：**
```json
{
  "logs": [
    {
      "id": "log_001",
      "actor": "刚子",
      "action": "task.create",
      "target": "task_abc123",
      "ip": "192.168.1.100",
      "timestamp": "2026-05-27T10:30:00Z",
      "result": "success"
    }
  ],
  "total": 1
}
```

### 3.3 Search 模块

#### GET /api/v1/search
全局搜索（跨 Goals/Projects/Tasks/Scenarios）。

**查询参数：**
- `q`（string，必填）：搜索关键词
- `type`（string，可选）：限定类型，如 `task`、`goal`、`scenario`
- `limit`（integer，可选）：返回数量，默认 20

**响应示例：**
```json
{
  "results": [
    {"type": "task", "id": "task_abc123", "title": "完成 API 文档", "score": 0.95},
    {"type": "goal", "id": "goal_xyz", "title": "Sprint 98 交付", "score": 0.85}
  ],
  "total": 2,
  "query": "API 文档"
}
```


```json
  "total_phases": 3,
```

## 5. 认证说明

除 `/api/v1/docs/*` 和 `/api/v1/health` 外，所有接口均需认证。认证方式：通过 HTTP Header 传递 `Authorization: Bearer <token>`。
