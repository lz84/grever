## 一、概述

### 1.1 接口范围

本章节定义能力标签（Capability Tags）的管理与查询接口。能力标签是调度引擎进行任务-智能体匹配的核心依据，支持标签的创建、查询、更新、删除操作，以及基于标签的任务推荐和 prerequisite 校验。

### 1.1 接口总表

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 创建标签 | POST | `/api/v1/industry-tags` | 创建新的行业能力标签 |
| 查询标签 | GET | `/api/v1/industry-tags` | 列表查询，支持行业/维度/状态筛选 |
| 查询标签详情 | GET | `/api/v1/industry-tags/{tag_id}` | 获取单个标签详情 |
| 更新标签 | PUT | `/api/v1/industry-tags/{tag_id}` | 更新标签信息，含状态变更 |
| 删除标签 | DELETE | `/api/v1/industry-tags/{tag_id}` | 软删除或硬删除 |
| 查询引用关系 | GET | `/api/v1/industry-tags/{tag_id}/references` | 查询标签被哪些实体引用 |
| 查询标签提供者 | GET | `/api/v1/industry-tags/{tag_id}/providers` | 查询提供某标签的技能/Agent |
| 校验 prerequisites | POST | `/api/v1/industry-tags/validate-prerequisites` | 批量校验标签 prerequisites |
| 推荐标签 | GET | `/api/v1/agents/{agent_id}/tag-recommendations` | 基于 Agent 画像推荐标签 |
| 标签统计 | GET | `/api/v1/industry-tags/_stats` | 获取标签使用统计 |
| 行业列表 | GET | `/api/v1/industry-tags/_industries` | 获取所有行业分类 |
| 按行业筛选 | GET | `/api/v1/industry-tags/_by-industry/{industry}` | 按行业获取标签列表 |
| Agent 标签推荐 | GET | `/api/v1/industry-tags/agent-tag-recommend` | Agent 标签推荐 |
| Agent 标签列表 | GET | `/api/v1/industry-tags/agent-tags` | 获取所有 Agent 标签 |

### 1.2 创建标签

```
POST /api/v1/industry-tags
Content-Type: application/json

Request Body:
{
  "tag_id": "chem:hazmat-identification",
  "tag_name": "危化品识别",
  "dimension": "professional",
  "level": "advanced",
  "description": "能够识别和分类各类危险化学品",
  "industry": "chemical-emergency",
  "prerequisites": ["chem:msds-parsing"],
  "tools": ["chemical-db-tool"],
  "examples": ["根据MSDS快速识别苯类危化品"]
}

Response 201:
{
  "tag_id": "chem:hazmat-identification",
  "tag_name": "危化品识别",
  "dimension": "professional",
  "level": "advanced",
  "status": "active",
  "version_major": 1,
  "version_minor": 0,
  "version_patch": 0,
  "created_at": 1716800000
}

Error Responses:
- 400: 标签ID不符合命名空间规则
  {"error": "invalid_tag_id", "detail": "标签ID必须符合格式 '行业前缀:能力标识符'"}
- 409: 标签ID已存在
  {"error": "tag_already_exists", "tag_id": "chem:hazmat-identification"}
```

### 1.5 更新标签

```
PUT /api/v1/industry-tags/{tag_id}
Content-Type: application/json

Request Body:
{
  "tag_name": "危化品识别（修订版）",
  "description": "能够识别和分类各类危险化学品，含复合危化品",
  "prerequisites": ["chem:msds-parsing", "chem:chemical-classification"],
  "tools": ["chemical-db-tool", "hazardous-level-assessor"]
}

Response 200:
{
  "tag_id": "chem:hazmat-identification",
  "tag_name": "危化品识别（修订版）",
  "version_change": "MINOR",
  "version": "1.1.0",
  "version_major": 1,
  "version_minor": 1,
  "version_patch": 0
}

Special: 状态变更
Request Body:
{
  "status": "deprecated",
  "replaced_by": "chem:hazmat-identification-v2"
}

Response 200:
{
  "tag_id": "chem:hazmat-identification",
  "status": "replaced_by",
  "replaced_by": "chem:hazmat-identification-v2",
  "reference_count": 8,
  "warning": "该标签被8个实体引用，更新前请确认是否需要迁移引用"
}

Error Responses:
- 400: replaced_by 指向不存在的标签
  {"error": "replacement_tag_not_found", "tag_id": "chem:xxx"}
```

### 1.9 校验 prerequisites

```
POST /api/v1/industry-tags/validate-prerequisites
Content-Type: application/json

Request Body:
{
  "tag_ids": ["chem:hazmat-identification"],
  "strict_mode": false
}

Response 200:
{
  "valid": false,
  "missing_prerequisites": {
    "chem:hazmat-identification": ["chem:msds-parsing"]
  },
  "circular_dependencies": [],
  "auto_add_tags": ["chem:msds-parsing"],
  "warnings": []
}

Response 200 (strict_mode=true 且有缺失):
{
  "valid": false,
  "missing_prerequisites": {
    "chem:hazmat-identification": ["chem:msds-parsing"]
  },
  "error": "prerequisites_not_satisfied",
  "detail": "标签 chem:hazmat-identification 缺少前置标签 chem:msds-parsing"
}

Response 200 (检测到环形依赖):
{
  "valid": false,
  "circular_dependencies": [["A", "B", "A"]],
  "error": "circular_dependency_detected"
}
```

## 三、错误码定义

| HTTP 状态码 | 错误码 | 说明 |
|------------|--------|------|
| 400 | `invalid_tag_id` | 标签ID不符合命名空间规则 |
| 400 | `invalid_custom_tag_id` | 定制标签ID不符合格式要求 |
| 400 | `custom_pack_requires_base` | 定制包未指定 base_pack_id |
| 400 | `prerequisites_not_satisfied` | 标签 prerequisites 不满足 |
| 400 | `circular_dependency_detected` | 检测到环形依赖 |
| 400 | `tag_conflict` | 定制标签与标准包冲突 |
| 400 | `replacement_tag_not_found` | 替换目标标签不存在 |
| 404 | `tag_not_found` | 标签不存在 |
| 404 | `pack_not_found` | 行业包不存在 |
| 404 | `invalid_base_pack` | base_pack_id 无效或非标准包 |
| 409 | `tag_already_exists` | 标签ID已存在 |
| 409 | `pack_already_exists` | 行业包名称已存在 |
