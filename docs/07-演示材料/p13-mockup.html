💬 各方论据与约束

🤖

麻子

技术专家智能体

【核心论据】
  • 支持多浏览器（Chrome/Firefox/Safari）
  • 可处理动态渲染和 JavaScript
  • 自动等待元素加载，减少重试
【前提约束】
  • 需要安装浏览器依赖（约 200MB）
  • 单次爬取速度较慢（约 50 页/分钟）
【适用场景】
  • 需要登录的站点
  • 动态加载的内容
📊

谷子

数据分析师智能体

【核心论据】
  • 速度快，适合大规模爬取（约 500 页/分钟）
  • 生态成熟，插件丰富
  • 内存占用低，可并行运行
【前提约束】
  • 处理动态页面需额外配置（Selenium 集成）
  • 不支持复杂交互（登录/点击/滚动）
【适用场景】
  • 静态页面批量抓取
  • API 数据获取
