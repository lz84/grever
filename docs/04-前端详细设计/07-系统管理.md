## 1. 智能体管理（/system/agents）

### 1.1 页面布局

```
┌────────────────────────────────────────────────────────────────┐
│ 🤖 智能体管理                                  [🔄 刷新]        │
│ 管理所有智能体的状态与能力                                       │
├────────────────────────────────────────────────────────────────┤
│ 智能体名称 │ 能力      │ 状态  │ 负载  │ 当前任务 │ 触发方式 │ 操作│
│ ────────────────────────────────────────────────────────────── │
│ 刚子      │ 编排/规划 │ 🟢在线│ 78%  │    1    │ 轮询    │ 详情  │
│ 谷子      │ 交易/分析 │ 🟢在线│ 62%  │    2    │ 轮询    │ 详情  │
│ 麻子      │ 编码/调试 │ 🟡繁忙│ 91%  │    3    │ 轮询    │ 详情  │
│ 蚊子      │ 内容/研究 │ 🟢在线│ 45%  │    2    │ 轮询    │ 详情  │
│ 扣子      │ 开发      │ 🔴离线│ 62%  │    0    │ 轮询    │ 详情  │
└────────────────────────────────────────────────────────────────┘
```

### 1.2 字段定义

| 字段 | 数据来源 | 显示格式 | 交互 |
|------|----------|----------|------|
| 智能体名称 | `agent.name` | 文本 + 代号：`刚子 (gangzi)` | 点击 → 打开详情 Modal |
| 能力 | `agent.capabilities` | 多个标签，最多 3 个 | 超出显示 `+N` |
| 状态 | `agent.status` | 圆点 + 文字：在线(绿)/繁忙(橙)/离线(红) | |
| 负载 | `agent.load` | 百分比 + 进度条 | >90% 红色，70-90% 橙色，<70% 绿色 |
| 当前任务 | `agent.current_tasks` | 数字 | 点击 → 跳转 `/coordination/tasks?agent={id}` |
| 触发方式 | `agent.trigger_mode` | 文本：轮询/SSE | |
| 操作 | — | [详情] 按钮 | 打开智能体详情 Modal |

### 1.3 状态映射

| API 值 | 显示 | 颜色 |
|--------|------|------|
| `idle` / `online` | 在线 | 绿色 |
| `busy` / `working` | 繁忙 | 橙色 |
| `offline` / 其他 | 离线 | 红色 |

**触发**：点击智能体名称 / [详情]

**布局**：

```

│ 负载：████████░░░░  78%                                        │
│ 当前任务：1                          [查看任务]                  │
│ 触发方式：轮询（每 10 秒）                                       │

├────────────────────────────────────────────────────────────────┤
│ ── 执行历史 ─────────────────────────────────────────────────── │
│ 任务名称    │ 状态  │ 耗时   │ 完成时间                         │

│ 共 24 条               [上一页] 1 2 3 [下一页]                  │
└────────────────────────────────────────────────────────────────┘
```

**字段定义**：

| 区块 | 字段 | 数据来源 | 显示格式 |
|------|------|----------|----------|
| 头部 | 名称 | `agent.name` | 大标题 |
| 头部 | Agent ID | `agent.id` | 括号内 |
| 头部 | 状态 | `agent.status` | 圆点 + 文字 |
| 头部 | 角色 | `agent.metadata.role` | 文本 |
| 头部 | 模型 | `agent.metadata.model` | 文本 |
| 负载 | 负载百分比 | `agent.load` | 进度条 + 百分比 |
| 负载 | 当前任务 | `agent.current_tasks` | 数字 |
| 负载 | 触发方式 | `agent.trigger_mode` | 文本 |
| 负载 | 轮询间隔 | `agent.poll_interval_seconds` | 文本 |
| 负载 | 注册时间 | `agent.registered_at` | `YYYY-MM-DD HH:mm` |
| 负载 | 最后心跳 | `agent.last_heartbeat` | `YYYY-MM-DD HH:mm` |
| 注册信息 | 地址 | `agent.address` | 文本 + 复制按钮 |
| 注册信息 | 元数据 | `agent.metadata` | JSON 格式化 |
| 执行历史 | 任务名称 | `GET /api/v1/tasks?agent_id={id}` | 文本 |
| 执行历史 | 状态 | `task.status` | ✅完成/❌失败 |
| 执行历史 | 耗时 | 计算 | `xmin` |
| 执行历史 | 完成时间 | `task.completed_at` | `MM-DD HH:mm` |

**交互**：

| 元素 | 触发行为 | 说明 |
|------|----------|------|
| [×] 关闭 | 关闭 Modal | |
| ESC 键 | 关闭 Modal | |
| 点击遮罩 | 关闭 Modal | |
| [📋 复制] | 复制地址到剪贴板 | 成功 Toast `已复制` |
| [查看任务] | 跳转 `/coordination/tasks?agent={id}` | |

### 1.5 API 调用

| 操作 | 方法 | 端点 | 说明 |
|------|------|------|------|
| 加载智能体列表 | GET | `/api/v1/agents` | |
| 执行历史 | GET | `/api/v1/tasks?agent_id={id}&limit=4` | |
| 复制地址 | `navigator.clipboard.writeText()` | 浏览器 API | |

## 4. 验收标准

| # | 验收项 | 测试方法 | 通过标准 |
|---|--------|----------|----------|
| 1 | 智能体列表加载 | 访问 `/system/agents` | 显示所有智能体 |

| 7 | 地址复制 | 点击 [📋 复制] | Toast `已复制` |

| 9 | 执行历史 | Modal 中查看 | 显示分页列表 |
| 10 | 团队页面 | 访问 `/system/teams` | 显示开发中提示 |
| 11 | 设置页面 | 访问 `/system/settings` | 显示开发中提示 |
