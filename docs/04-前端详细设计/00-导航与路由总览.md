# 导航与路由总览

> 本文档定义 8 大模块的整体导航结构、路由规范、公共组件规范。
> 8 个子模块的详细设计见对应文档。

---

## 1. 模块总览

| 序号 | 模块 | 对应 | 核心职责 | 路由前缀 |
|------|------|------|----------|----------|
| 1 | 工作台 | 入口 | 全局概览、快速入口 | `/` |
| 2 | 协同中心 | 御(Reins) | 目标/项目/任务/执行的协同调度 | `/coordination` |
| 3 | 认知中心 | 悟(Grasp) | 知识库、认知评估、注入管理 | `/cognitive` |
| 4 | 安全中心 | 鉴(Vigil) | 监控、审计、告警 | `/security` |
| 5 | 场景库 | 达(Reach) | 场景实例化、数据背书、反馈回收 | `/scenarios` |
| 6 | 技能库 | 化(Evo) | 单体能力进化、Capsule、进化轨迹 | `/skills` |
| 7 | 系统管理 | — | 智能体管理、团队、设置 | `/system` |
| 8 | 可视化 | — | 看板、Trace 时间线、报表 | `/visual` |

## 2. 导航结构

### 2.1 侧边栏设计

**布局**：左侧垂直导航，可折叠

```
┌─────┬──────────────────────────┐
│ 🔷  │ 智能体协同驾驭平台        │  ← 顶部 Logo 区（折叠时只显示图标）
├─────┼──────────────────────────┤
│ 🏠  │ 工作台                    │  ← 模块 1
│ 🔗  │ 协同中心                  │  ← 模块 2
│ 🧠  │ 认知中心                  │  ← 模块 3
│ 🛡️  │ 安全中心                  │  ← 模块 4
│ 📋  │ 场景库                    │  ← 模块 5
│ 🧬  │ 技能库                    │  ← 模块 6
│ ⚙️  │ 系统管理                  │  ← 模块 7
│ 📊  │ 可视化                    │  ← 模块 8
├─────┼──────────────────────────┤
│ [⊞] │ 折叠/展开按钮             │  ← 底部
```

**展开宽度**：240px | **折叠宽度**：72px

### 2.2 导航项定义

| 序号 | 显示名 | 图标 | 路由 | 子路由 | 说明 |
|------|--------|------|------|--------|------|

### 2.3 活跃状态判断

```typescript
const isActive = (modulePrefix: string, currentPath: string) => {
  if (modulePrefix === '/') return currentPath === '/';
  return currentPath.startsWith(modulePrefix);
};
```

**规则**：
- 精确匹配优先（工作台 `/`）
- 前缀匹配其余模块（如 `/coordination/goals` 匹配 `/coordination`）

**位置**：主内容区顶部，页面标题上方

**格式**：`模块名 / 子模块 / 当前页面`

**各页面面包屑**：

| 页面 | 面包屑 |
|------|--------|

### 4.2 全局搜索

**位置**：顶部栏右侧

**快捷键**：`Ctrl+K`（Windows/Linux）/ `Cmd+K`（Mac）

**搜索范围**：所有模块的名称类字段（目标名称、项目名称、场景名称、技能名称、智能体名称等）

**搜索结果分组**：

| 分组 | 图标 | 最大显示 | 跳转 |
|------|------|----------|------|
| 目标 | `Target` | 5 | `/coordination/goals/:id` |
| 项目 | `FolderKanban` | 5 | `/coordination/projects/:id` |
| 任务 | `ListTodo` | 5 | `/coordination/tasks` |
| 场景 | `FileText` | 5 | `/scenarios/:id` |
| 技能 | `Dna` | 5 | `/skills/:id` |
| 智能体 | `Bot` | 5 | `/system/agents` |

**防抖**：300ms

**无结果**：显示 `未找到匹配结果`

### 4.4 空状态

**统一样式**：居中，图标 + 文案 + 操作按钮

```
┌──────────────────────┐
│       [图标]         │
│  暂无数据            │
│  [操作按钮]          │
└──────────────────────┘
```

**文案规范**：
- 列表为空：`暂无{模块名}，点击「{操作}」创建`
- 搜索无结果：`未找到匹配的{模块名}`
- 加载中：骨架屏（5 行占位）

### 4.5 错误处理

| 场景 | 显示 | 操作 |
|------|------|------|
| API 404 | 红色横幅：`无法连接后端服务，请检查服务器是否运行` | [重试] |
| API 500 | 红色横幅：`服务器错误：{error.message}` | [重试] |
| API 超时 | 红色横幅：`请求超时，请稍后重试` | [重试] |
| 网络断开 | 顶部全局提示：`网络连接已断开` | 自动重试 |

### 4.6 分页规范

| 配置 | 值 |
|------|------|
| 默认每页 | 10 条 |
| 可选 | 10 / 20 / 50 |
| 控件位置 | 表格底部右侧 |
| 格式 | `共 {total} 条` + `[上一页] [页码] ... [下一页]` |
| 页码超出 | 超过 5 页时显示 `1 2 3 ... N` |
