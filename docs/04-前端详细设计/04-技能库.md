# 技能库 - 详细设计

> 路由前缀：`/skills`
> 对应：化(Evo) — 单体智能体能力进化、技能库、Capsule 市场、进化轨迹
> 子页面：技能列表、技能详情、Capsule 市场、进化轨迹

## 1. 技能列表（/skills）

### 1.1 页面布局

```
┌────────────────────────────────────────────────────────────────┐
│ 🧬 技能库                                        [🔍 搜索...]   │
│ 智能体能力管理                                                  │
├────────────────────────────────────────────────────────────────┤
│ [全部智能体▼] [全部类型▼]                        [重置筛选]     │
├────────────────────────────────────────────────────────────────┤
│ 技能名称     │ 所属智能体 │ 类型     │ 等级 │ 使用次数 │ 状态  │
│ ────────────────────────────────────────────────────────────── │
│ 灾情评估     │ 刚子       │ 分析     │ Lv.3│ 12次    │ 活跃  │
│ 预案匹配     │ 谷子       │ 研判     │ Lv.2│ 8次     │ 活跃  │
│ 资源调度     │ 麻子       │ 协调     │ Lv.4│ 15次    │ 活跃  │
│ 代码生成     │ 麻子       │ 编码     │ Lv.5│ 42次    │ 活跃  │

│ 共 20 条技能                      [上一页] 1 2 [下一页]        │
└────────────────────────────────────────────────────────────────┘
```

### 1.2 字段定义

| 字段 | 数据来源 | 显示格式 | 交互 |
|------|----------|----------|------|
| 技能名称 | `skill.name` | 文本，最多 20 字 | 点击 → `/skills/:id` |
| 所属智能体 | `skill.agent_id` 映射 | 智能体名称 | 点击 → `/system/agents` |
| 类型 | `skill.type` | 标签：分析(蓝)/研判(绿)/协调(橙)/编码(紫)/通用(灰) | |
| 等级 | `skill.level` | `Lv.x` (1-5) | |
| 使用次数 | `skill.usage_count` | 数字 + `次` | |
| 状态 | `skill.status` | 标签：活跃(绿)/已废弃(灰) | |

### 1.3 筛选

| 筛选项 | 选项 | 说明 |
|--------|------|------|
| 所属智能体 | 全部 + 每个智能体 | |
| 类型 | 全部/分析/研判/协调/编码/通用 | |
| 搜索 | 输入关键词 | 300ms 防抖 |

### 1.4 API 调用

| 操作 | 方法 | 端点 | 说明 |
|------|------|------|------|
| 加载技能列表 | GET | `/api/v1/skills` | 待实现 |
| 加载智能体列表 | GET | `/api/v1/agents` | 用于筛选 |

## 6. API 返回结构定义

### 6.1 技能列表 API

**端点**：`GET /api/v1/skills`

**查询参数**：
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `agent_id` | string | ❌ | 按智能体过滤 |
| `type` | string | ❌ | 按类型过滤 |
| `q` | string | ❌ | 搜索关键词 |
| `limit` | number | ❌ | 默认 20 |
| `offset` | number | ❌ | 默认 0 |

**返回结构**：
```typescript
interface SkillResponse {
  total: number;
  items: Skill[];
}

interface Skill {
  id: string;               // 技能 ID
  name: string;             // 技能名称
  agent_id: string;         // 所属智能体 ID
  type: 'analysis' | 'judgment' | 'coordination' | 'coding' | 'general';
  level: number;            // 1-5
  usage_count: number;      // 使用次数
  status: 'active' | 'deprecated';
  description: string;      // 技能描述
  metrics: {
    accuracy: number;       // 0-100
    speed: number;          // 0-100
    stability: number;      // 0-100
  };
  created_at: string;       // ISO 8601
  updated_at: string;       // ISO 8601
}
```

**错误处理**：
| HTTP 状态码 | 返回 | 前端处理 |
|-------------|------|----------|
| 400 | `{ "error": "Invalid type" }` | 显示错误 Toast |
| 500 | `{ "error": "Internal error" }` | 红色横幅 + [重试] |

### 6.2 Capsule 安装/卸载 API

**端点**：`POST /api/v1/capsules/{id}/install`

**请求体**：
```typescript
interface CapsuleInstallRequest {
  target_agent_id?: string;  // 安装到哪个智能体（可选，不填则安装到当前用户）
  version?: string;          // 指定版本（可选，不填则安装最新）
}
```

**返回结构**：
```typescript
interface CapsuleInstallResponse {
  success: boolean;
  capsule_id: string;
  installed_version: string;
  installed_at: string;  // ISO 8601
  message?: string;      // 成功提示或警告
}
```

**错误处理**：
| HTTP 状态码 | 返回 | 前端处理 |
|-------------|------|----------|
| 404 | `{ "error": "Capsule not found" }` | Toast `Capsule 不存在` |
| 409 | `{ "error": "Already installed" }` | 按钮变为 [已安装] |
| 400 | `{ "error": "Incompatible agent" }` | Toast `与目标智能体不兼容` |

**卸载 API**：`POST /api/v1/capsules/{id}/uninstall`
- 无需请求体
- 返回 `{ "success": boolean, "message": string }`

### 6.3 进化记录 API

**端点**：`GET /api/v1/evolution-logs`

**查询参数**：
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `agent_id` | string | ❌ | 按智能体过滤 |
| `skill_type` | string | ❌ | 按技能类型过滤 |
| `limit` | number | ❌ | 默认 50 |

**返回结构**：
```typescript
interface EvolutionLogResponse {
  total: number;
  items: EvolutionLog[];
}

interface EvolutionLog {
  id: string;
  agent_id: string;
  agent_name: string;
  skill_name: string;
  skill_type: string;
  event_type: 'level_up' | 'metric_improved' | 'new_skill' | 'skill_deprecated';
  description: string;      // 如 "代码审查技能从 Lv.4 升级到 Lv.5"
  old_value: number;        // 旧值
  new_value: number;        // 新值
  created_at: string;       // ISO 8601
}
```

## 5. 验收标准
