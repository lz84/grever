### 1.2 字段定义

| 区块 | 字段 | 数据来源 | 显示格式 |
|------|------|----------|----------|
| 目标完成率 | 已完成数 | `GET /api/v1/goals?status=completed` 长度 | 数字 |
| 目标完成率 | 总数 | `GET /api/v1/goals` 长度 | 数字 |
| 目标完成率 | 完成率 | 计算 `已完成/总数` | 进度条 + 百分比 |
| 任务状态分布 | 各状态数量 | `GET /api/v1/tasks` 分组计数 | 进度条 + 百分比 + 数字 |
| 智能体负载 | 负载百分比 | `agent.load` | 进度条 + 百分比 |
| 智能体负载 | 任务数 | `agent.current_tasks` | 数字 |
| 执行趋势 | 每日执行数 | `GET /api/v1/workflows?since=7d` 按天分组 | 柱状图 |

|------|------|------|------|

| 智能体数据 | GET | `/api/v1/agents` | |

### 2.2 字段定义

| 字段 | 数据来源 | 显示格式 | 说明 |
|------|----------|----------|------|
| 执行名称 | `workflow.name` | 大标题 | |
| 执行时间 | `workflow.started_at` | `YYYY-MM-DD HH:mm` | 作为分组标题 |
| 步骤 | `workflow.steps` | 时间线节点 | |
| 步骤名称 | `step.name` | 文本 | |
| 步骤执行者 | `step.agent_id` | 智能体名称 | |
| 步骤状态 | `step.status` | ✅完成/🔄运行中/⏳等待/❌失败 | |
| 步骤耗时 | `step.completed_at - step.started_at` | `xmin` | |
| 冲突事件 | `disputes` 关联 | ⚠️ 冲突告警 | 显示在对应步骤下 |

### 2.3 交互

| 元素 | 触发行为 | 说明 |
|------|----------|------|
| 执行名称 | 展开/折叠 | 默认展开最近 3 条 |
| 冲突事件 | 点击 → 打开冲突处理 Modal | |
| 步骤 | 点击 → 跳转执行详情 | |

### 2.4 筛选

| 筛选项 | 说明 |
|--------|------|
| 时间范围 | 今天/最近 7 天/最近 30 天/自定义 |
| 执行 | 下拉选择特定执行 |
| 搜索 | 输入关键词 |

```

```

### 3.2 字段定义

| 区块 | 字段 | 数据来源 | 显示格式 |
|------|------|----------|----------|
| 报表列表 | 报表名称 | `workflow.name` | 文本 |
| 报表列表 | 执行时间 | `workflow.started_at` | `MM-DD` |
| 报表列表 | 总耗时 | 计算 | `xmin` |
| 报表列表 | 完成率 | 计算 `完成步骤/总步骤` | 百分比 |
| 报表列表 | 冲突解决 | 关联 `disputes` 数量 | 数字 |
| 报表详情 | 各字段 | 报表 API 返回 | 详细文本 |

### 3.3 交互

| 元素 | 触发行为 | 说明 |
|------|----------|------|
| 报表行 | 展开详情 | 手风琴展开 |
| [查看] | 展开详情 | 同上 |
| [导出 PDF] | 生成 PDF 文件 | 待实现 |
| [生成完整报告] | 打开报告 Modal | 包含完整 Trace + 智能体评估 + 冲突分析 |

### 3.4 API 调用

| 操作 | 方法 | 端点 | 说明 |
|------|------|------|------|
| 加载报表列表 | GET | `/api/v1/reports` | |
| 加载单个报表 | GET | `/api/v1/reports/{workflow_id}` | |
| 导出 PDF | GET | `/api/v1/reports/{workflow_id}/export?format=pdf` | 待实现 |

## 4. 验收标准

| # | 验收项 | 测试方法 | 通过标准 |
|---|--------|----------|----------|
| 1 | 看板加载 | 访问 `/visual/dashboard` | 显示统计卡片 + 图表 |
| 2 | 目标完成率 | 查看进度条 | 百分比正确 |
| 3 | 任务状态分布 | 查看分布图 | 各状态数量正确 |
| 4 | 智能体负载 | 查看负载条 | 百分比与智能体管理页一致 |
| 5 | 执行趋势 | 查看柱状图 | 数据按天分组 |
| 6 | Trace 查看 | 访问 `/visual/traces` | 显示时间线 |
| 7 | Trace 展开/折叠 | 点击执行名称 | 正确展开/折叠 |
| 8 | 报表列表 | 访问 `/visual/reports` | 显示报表 |
| 9 | 报表详情 | 点击报表行 | 展开详情 |
| 10 | 导出 PDF | 点击 [导出 PDF] | 生成 PDF（或提示开发中）|
| 11 | 空状态 | 无数据 | 友好提示 |
